Модуль 1: Вступ до Unity

У цьому модулі ви ознайомитесь з основними інструментами та можливостями Unity для створення 2D ігор:

1. Знайомство з Unity Editor

Unity Editor — це середовище розробки для створення ігор. Тут ви зможете бачити сцену, налаштовувати об’єкти, працювати з кодом і запускати проєкт.

2. Створення сцен та об'єктів

Ви дізнаєтесь, як створювати 2D сцену, додавати примітивні об'єкти, працювати з шарами та позиціювати елементи.

3. Робота зі спрайтами

Спрайти — це зображення для 2D графіки. Ви навчитесь імпортувати їх, розміщувати в сцені та створювати анімації.

4. Побудова простого ігрового рівня

Після засвоєння основ ви створите простий рівень з платформами, героєм та фоном, які зроблять гру живою.

5. Компіляція проєкту

Навчитесь запускати анімація для нашого середовища.

6. Створення анімації спокою та бігу

У цьому відео я покажу, як зробити анімацію спокою та бігу для персонажа при умові, що у нас декілька кадрів на одному скріні.
