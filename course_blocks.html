Головна Про нас Контакти Реєстрація

Модуль 1: Вступ до Unity

У цьому модулі ви ознайомитесь з основними інструментами та можливостями Unity для створення 2D ігор:

  • Знайомство з Unity Editor
  • Створення простих сцен та об'єктів
  • Основи роботи з 2D спрайтами

Модуль 2: Рух гравця

У цьому модулі ми розглянемо, як створити персонажа та додати йому фізику:

  • Створення персонажа та його рух
  • Робота з фізичними властивостями
  • Обробка стрибків та перевірка на землю

Модуль 3: Взаємодія з об’єктами

Цей модуль надасть знання для створення взаємодії з об'єктами в грі:

  • Збирання монет та предметів
  • Створення анімації болю
  • Рух по драбині без анімацій – просте рішення
  • Додавання музика на сцену
  • Стрільба і вбивство ворогів
  • Переміщення по драбині
  • Рандомна зміна кольору обєктів

Модуль 4: Анімації та звук

Створення анімацій та додавання звуків до гри:

  • Створення ігрового рівня з фоном, NPC і колайдерами
  • Платформа, що зникає після стрибка та відтворення звуку
  • Створення NPC ворог з атакою, ходьбою та анімацією
  • Додавання звукових ефектів
  • Система життя гравця
  • UI Індикатор життя
  • Створення анімації в Animator
  • Анімація ворогів
  • Додавання Звуку у 2D-грі
  • Створення телепорту

Модуль 5: Рівні гри

Створення кількох рівнів та переходів між ними:

  • Робота з різними сценами
  • Перехід між рівнями
  • Завантаження та перезапуск гри
  • Checkpoint збереження місця розміщення
  • Сторення та перехід між сценами (рівнями гри) використовуючи Loading Screen

Модуль 6: Місії

Створення кількох місій (завдань) та переходів між ними:

  • Вступ до квестів
  • Простий квест «Принеси предмет»
  • Квест з NPC (діалог + завдання)
  • Ланцюжок квестів
  • Квест з умовою бою
  • Квест з таймером
  • Комбіновані квести
  • Система відстеження квестів (Quest Manager)

Модуль 7: Створення NPC та ReidBoss

  • Простий ворог (рух + атака)
  • Штучний інтелект ворога (патрулювання, переслідування гравця)
  • Дружній NPC (рух, базова взаємодія)
  • NPC із діалогами (простий діалоговий скрипт)
  • Рейдбосс (механіка атак)
  • Рейдбосс з кількома фазами бою
  • Анімації ворогів і босів (атака, отримання урону, смерть)
  • Система дропів (лут після перемоги над ворогом чи босом)
  • Комбіновані бої (гравець + NPC проти рейдбосса)

Модуль 8: Інтерфейс гри

Створення меню та екранів гри:

  • Меню старту та паузи
  • Екран Game Over та Win
